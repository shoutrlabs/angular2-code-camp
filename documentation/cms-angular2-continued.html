Directives

Directives allow you to manipulate DOM elemements

MAC: 

MAC: <input type="text" mask="##:##:##:##:##:##">

Directives

import { Directive } from '@angular/core';

@Directive({
    selector: 'input[mask]',
    host: {
        '(keyup)': 'transform($event)'
    }
})
export class InputMaskDirective {
    transform(event) {
        const newVal = this.maskInput(event.target.value);
        if(newVal && event.target.value !== newVal) event.target.value = newVal;
    }
    maskInput(maskMe) { // this is simple to test!
        ...
    }
}

Component & Directive: host

With host we can access the hosting DOM element.

host: {
    '(keyup)': 'transform($event)'
}

is like writing

<input type="text"(click)="transform($event)"mask="##:##:##:##:##:##">

only that we cannot do it because the transform function would belong to the parent component.

Pipes

Pipes allow us to transform variables in templates

<input 
    type="text" 
    placeholder="... filter users" 
    [(ngModel)]="query"/>

<div *ngFor="let user of users | filter:query">
    {{user.firstName}}
</div>

Pipes

import { Pipe } from '@angular/core';
@Pipe({ name: 'filter' })
export class FilterArrayOfObjectsPipe {
    transform(value, query) {
        query = query || '';
        if (!query) { return value; // empty query, don't filter
        } else if (value) {
            return value.filter( item => // filter((item) => Boolean)
                Object.keys(item).filter( key =>  // expects item to be object
                    (typeof item[key] === 'string') && // value String?
                    (item[key].toLowerCase().indexOf(query.toLowerCase()) !== -1)
                ).length > 0 
            );
        }
    }
}

Register filter and pipes

Don't forget to register your filters and pipes in your Angular module just like componets.

import { NgModule } from '@angular/core';
import { FilterArrayOfObjectsPipe } from './filter.pipe';
import { InputMaskDirective } from './input-mask.directive';

@NgModule({
    declarations: [
        FilterArrayOfObjectsPipe,
        InputMaskDirective,
    ],
    …
})
export class MyModule { }

Services

import { Injectable } from '@angular/core';

@Injectable()
export class UsersService {
    constructor() {
        this.users = [];
    }
    add(user) {
        this.users.push(user)
    }
}

Services are singletons - only one class instances - that implement logic

Services: dependency injection

import { Component } from '@angular/core';
import { UsersService } from './users.service';

@Component({
    // selector omitted, we insert this component with the router
    template: `
<ul>
    <li *ngFor="let user of users">{{user.firstName}}</li>
</ul>
    `,
})
export class UserListComponent {
    static get parameters() {
        return [[UsersService]];
    }
    constructor(usersService) {
        this.users = usersService.users;
    }
}

Services: provider

The example before will fail because the UsersService must be instanciated before.

This is done in an Angular Module

import { NgModule } from '@angular/core';
import { UsersService } from './users.service';

@NgModule({
    providers: [
        UsersService,
    ],
    …
})
export class UsersModule { }

Shared Modules

We don't want to import our filters and pipes all the time, so we create a shared module.

import { NgModule } from '@angular/core';
import { FilterArrayOfObjectsPipe } from './filter.pipe';
import { InputMaskDirective } from './input-mask.directive';

@NgModule({
    declarations: [
        FilterArrayOfObjectsPipe,
        InputMaskDirective,
    ],
    exports: [
        FilterArrayOfObjectsPipe,
        InputMaskDirective,
    ]
})
export class TemplateHelpersModule { }

Shared Modules

import { UserListComponent } from './user-list.component';
import { TemplateHelpersModule } from './template-helpers.module';

@NgModule({
    imports: [
        TemplateHelpersModule,
    ],

        UserListComponent,
    ],
    …
})

Router: root module setup

import { RouterModule } from '@angular/router';
import { UsersModule } from './users/users.module';
import { AppComponent } from './app.component';

const routes = [
    { path: '**', redirectTo: '/users' },
];

@NgModule({
    imports: [
        BrowserModule,
        RouterModule.forRoot(routes),
        UsersModule,
    ],
    declarations: [AppComponent],
    bootstrap: [AppComponent],
})
export default class { }

Achtung!

  • do not forget to import your component, directive, pipe in a module
  • use shared modules for components, directives, … you want to reuse
  • do not redeclare a component, directive, etc. in more than one module
