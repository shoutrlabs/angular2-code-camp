# Code Camp Project: 

## Installation

First install yarn globally
```
npm i -g yarn
```
then install packages specified in `package.json`.
```
yarn
```

## Building

Build icons with 
```
npm run icons
```

Build documentation with 
```
npm run docu
```

## Testing
Run test with 
```
npm test
```

Watch files for changes and run tests with
```
npm run watch:test
```
