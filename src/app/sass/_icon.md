Icons
=======

All icons were generated with icomoon.

The current icomoon export is here: <a href="../resources/shoutrlabs.zip">shoutrlabs.zip</a>. If you add new icons please overwrite this file and check it into the git.

To update the fonts go to https://icomoon.io/app/#/select import the `selection.json` file in the zip and generate the new font. Download the `shoutrlabs.zip` to `/resources` and run `npm run icons`.

All icons eding with `_L` will generate a large icon all other small icons.
